# AI Growth Platform Demo

版本：V6 策略中心评审版

本仓库用于存放 AI 增长平台「回传策略中心」交互 demo。

## Demo 入口

- 本地入口：`index.html`
- 原型文件：`prototype/回传策略中心_交互原型_v6.html`

## 本版本范围

- 策略管理
- 运行详情
- 回传记录
- 回传配置
- 主线阶段与分支策略配置
- 回传筛选、扣量、平台转化类型配置

## 预览方式

从仓库根目录启动本地服务：

```bash
python3 -m http.server 8765
```

打开：

```text
http://localhost:8765/
```
