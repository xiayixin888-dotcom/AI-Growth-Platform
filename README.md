# AI Growth Platform Demo

版本：V6 策略中心评审版 + 数据面板 V1.0 草稿

本仓库用于存放 AI 增长平台交互 demo 和 PRD 草稿。

## Demo 入口

- 本地入口：`index.html`
- 回传策略中心：`prototype/回传策略中心_交互原型_v6.html`
- 投放回传数据面板：`prototype/投放回传数据面板_交互原型.html`

## 本版本范围

### 回传策略中心

- 策略管理
- 运行详情
- 回传记录
- 回传配置
- 主线阶段与分支策略配置
- 回传筛选、扣量、平台转化类型配置

### 投放回传数据面板

- 数据面板首页
- 广告对象分析
- 落地页分析
- 策略组执行分析
- 监控提醒
- 小时级分析

## PRD 文档

- 投放回传数据面板 PRD：`投放回传数据面板_PRD.md`

## 预览方式

从仓库根目录启动本地服务：

```bash
python3 -m http.server 8765
```

打开：

```text
http://localhost:8765/
```
